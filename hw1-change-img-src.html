Homework 1: HTML

Estimated Time

1 - 2 days

Description

  1. You will modify your first HTML file by adding a picture of yourself

Instructions

  1. First, download an HTML editor (for Windows I recommend Notepad++ for MacOS I recommend CotEditor)
  2. Now open this file in your HTML editor and change the src of the image at the end of this file to a picture of yourself
  3. You can get the URL of any picture you find online by right clicking on the image then selecting Copy Image Address
  4. Once you have done that, open the file in your browser and check to see your image displays correctly
  5. Send the HTML file back to me on Slack. Good luck and have fun!
