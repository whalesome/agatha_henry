Homework 6: Free Code Camp

Estimated Time

1 - 3 days

Description

  • In this homework, you will register with Free Code Camp
  • One of the main goals of the Diginido Coding School program is to achieve the Free Code Camp Front End Development Certificate.

Instructions

  • Goto https://www.freecodecamp.com and register with the github account you created in hw4
  • Begin with Say Hello to HTML elements and go through and finish up to Comment Out HTML
  • Then, open the file freecodecamp.html which should be in your repository
  • Pick a hero of yours and replace the HTML title with his name.
  • Add an <h1> element with his name and a <p> element with a short description of your hero. Make sure these are in the <body> element.
  • Commit the file once you are done.
  • Good luck!
