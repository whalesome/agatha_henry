Homework 5: Adding a file to Github

Estimated Time

1 - 2 days

Description

In this homework we will be adding a file to github. This is useful so that other developers you are working with can read any changes you write to any file in whatever project you are working on.

Instructions

Copy this file hw5-github-source-tree-write.html into that folder

Go into SourceTree, Click on Uncommitted changes

You should see hw5-github-source-tree-write.html under Unstaged files, check it

Click Commit on the top left of SourceTree

In the box below write a description like "hw5!"

Check "Push changes immediately to origin/master"

Press Commit on the bottom right

Wait for SourceTree to finish committing to github

When that's done, you've successfully uploaded your first file. Let me know on Slack. Congrats!
